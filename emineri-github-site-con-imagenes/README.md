# EMINERI GitHub Site

Sitio web estático inspirado en la estructura de `emineri.com`, listo para subir a GitHub Pages o a cualquier hosting estático.

## Archivos

- `index.html`: contenido principal del sitio.
- `styles.css`: estilos responsivos, colores, tarjetas y layout.
- `script.js`: selector de idioma ES/EN.

## Cómo publicarlo en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube estos tres archivos a la raíz del repositorio.
3. Entra en `Settings > Pages`.
4. En `Build and deployment`, selecciona `Deploy from a branch`.
5. Elige la rama `main` y la carpeta `/root`.
6. Guarda los cambios y espera a que GitHub genere la URL.

## Personalización rápida

- Cambia el correo en `index.html` si necesitas otro destino de contacto.
- Edita los textos dentro de `index.html` y `script.js` para ajustar la versión en español e inglés.
- Reemplaza los bloques visuales de `styles.css` por imágenes propias si tienes fotos o logo autorizado.
